# DesktopVoiceAssistant

## Features
- Greet according to day time
- Perform google search
- Find location on google maps
- Send emails
- Wikipedia search
- Play offline music
- Open Spotify
- Open youtube

## Package Used 
- pyttsx3 (text-to-speech)
- datetime
- speech_recognition
- wikipedia
- os
- webbrowser
- smtplib
